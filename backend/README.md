# Development


Configura variables example:

```
mkdir db;

dev:

PORT=8080
DBTYPE="memory"
DBPATH=""

test:

PORT=8081
DBTYPE="file"
DBPATH="db/store_test.json"
```

# Run dev

```

npm run dev

```

# Run test

```
mkdir db

npm run test

```
